# 🌆 Wallzify

Wallzify is a beautifully designed Flutter application that offers a vast collection of high-quality wallpapers across a wide range of categories. Whether you love minimal aesthetics, anime art, abstract visuals, or nature photography, Wallzify provides an easy, smooth, and fast experience to explore, preview, and set wallpapers with a single tap.

## ✨ Features

- 🖼️ **High-Quality Wallpapers** – Browse thousands of curated HD & 4K wallpapers.
- 🔍 **Search Functionality** – Quickly find wallpapers by keyword or category.
- 🌈 **Categories & Tags** – Discover wallpapers sorted into themes like Nature, Anime, Abstract, Space, and more.
- ❤️ **Favorites** – Save your favorite wallpapers to access them offline.
- 📲 **One-Tap Set** – Easily set wallpapers to home screen, lock screen, or both.
- ⬇️ **Downloads** – Download wallpapers directly to your device.
- 🌙 **Dark Mode** – Beautiful UI with light and dark theme support.
- 🧠 **Smart Suggestions** – Personalized wallpaper suggestions (future feature).

## 🛠️ Tech Stack

- **Flutter** – Cross-platform UI toolkit.
- **Dart** – Programming language used.
- **Firebase** – For backend services (optional).
- **REST API** – To fetch wallpapers dynamically.

## 📦 Installation

1. Clone the repo:
   ```bash
   git clone https://github.com/your-username/wallzify.git
   cd wallzify
   ```

2. Install dependencies:
   ```bash
   flutter pub get
   ```

3. Run the app:
   ```bash
   flutter run
   ```

## 📁 Folder Structure

```
lib/
```
